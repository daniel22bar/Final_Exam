this service will return information about games.

if you want to get information about suspicious activity
you need to provided a start date in the next format: 2014-01-29T10:48:17-03:00
you need to provided a end date in the next format: 2020-03-22T08:21:52-02:00
example url:
http://localhost:8080/game/suspicious?startDate=2014-01-29T10:48:17-03:00&endDate=2020-03-22T08:21:52-02:00

if you want to get statistic information about specific game you need to provide:
 game name = monte
 start date in the next format: 2014-01-29T10:48:17-03:00
 end date in the next format: 2020-03-22T08:21:52-02:00
example url:
http://localhost:8080/game/gameStatistic?gameName=monte&startDate=2014-01-29T10:48:17-03:00&endDate=2020-03-22T08:21:52-02:00

if you want to get statistic information about all games you need to provide:
 start date in the next format: 2014-01-29T10:48:17-03:00
 end date in the next format: 2020-03-22T08:21:52-02:00
example url:
http://localhost:8080/game/allGameStatistic?startDate=2014-01-29T10:48:17-03:00&endDate=2020-03-22T08:21:52-02:00
